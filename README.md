# Random-Password-Generator

## Description

This project a random password generator, that uses javascript to create a random password based on user input and preference!

## Usage

Click on the "Generate Password" button to start the  process, in which a popup window will appear and will ask for your preference on the length of the password. The password must be 8 or more characters, but the maximum character limit is 128. 
A series of questions about uppercase, lowercase, numbers, and special characters will follow in which you are free to include or not include them in your password. 
After answering the questions, your randomized password will be displayed in the text box based on your input to the popup questions.
